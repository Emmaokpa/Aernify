# **App Name**: Earnify

## Core Features:

- HTML5 Game Integration: Integrate HTML5 games from Playgama for users to play and earn rewards.
- Offer Wall Integration: Allow users to perform affiliate and CPA offers to earn in-app currency. User sees a list of possible offers. When an offer is clicked, the app acts as a tool to confirm whether or not the requirements for the offer have been completed.
- Shop Section: Implement a shop section where users can purchase tech gadgets and watches using in-app currency or real money.
- Gift Card Redemption: Enable users to redeem their earned in-app coins for various gift cards.
- Daily Login & Referral Bonuses: Reward users with daily login bonuses and referral bonuses for inviting new users.
- Leaderboard & Weekly Prizes: Display a leaderboard showcasing top users and award weekly prizes to the winners.
- VIP Subscription: Offer a VIP subscription for $9.99, providing subscribers with 2x earning rates.

## Style Guidelines:

- Primary color: Deep purple (#4B0082), emulating the image, creating a premium, mysterious and engaging feel.
- Background color: Dark gray (#212121) to enhance contrast and readability.
- Accent color: Bright Yellow (#FFD700) analogous to the primary but clearly distinct in value, used for calls to action, highlights, and interactive elements, ensuring they stand out. Mimics color scheme in provided image.
- Body and headline font: 'Inter', sans-serif, chosen for its clean and modern appearance, suitable for both headlines and body text, ensures readability across all devices.
- Use flat, vector-based icons with a consistent style that complements the app's theme.
- The app UI should mirror the image.
- Implement subtle animations and transitions to provide a smooth and engaging user experience.
